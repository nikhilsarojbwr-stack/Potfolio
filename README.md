Inclusive and universally usable by design. Large text, high contrast, clear navigation. Designed for users with varying abilities—color isn't the only way to communicate. Feels solid and dependable. Works perfectly on all devices and with all assistive technologies.

**LAYOUT SYSTEM:**  
Clean grid layout with very clear sections. Large touch targets (minimum 44px × 44px). Skip navigation links visible. Breadcrumb navigation throughout. Consistent navigation across all pages. Form fields with clear labels and error messages. Headings follow proper H1-H6 hierarchy throughout.

**TYPOGRAPHY:**  
Sans-serif throughout (Inter, Open Sans). Minimum 16px base font size. 1.5 line-height minimum. Variable font weights for hierarchy. No text over background images without high-contrast overlays. Readable monospace for code/technical content. Letter-spacing slightly increased for readability.

**COLOR PALETTE:**  
High contrast ratios: WCAG AAA compliant minimum (7:1 for body text, 4.5:1 for large text). Primary: Dark blue (#003366) or black (#000000). Accent: Orange (#ff6b35) or red (#e63946) that passes contrast checks. No color-only information (always backed by pattern, text, or icon).

**MOTION & INTERACTIONS:**  
Respects `prefers-reduced-motion` setting completely. Focus states are highly visible (minimum 3px outline). Keyboard navigation is perfect—every element is tab-able. No auto-playing audio/video. Animations are purposeful and optional. Transitions are smooth (200-300ms).

**BEST FOR:**  
Government websites, healthcare systems, public services, enterprise software for diverse users, accessibility-first companies, educational platforms.

**TONE:**  
Inclusive, trustworthy, clear, helpful, professional, welcoming.
